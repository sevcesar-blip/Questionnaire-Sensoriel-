# Questionnaire Sensoriel - Sensations Fun !

Bienvenue sur le **Questionnaire Sensoriel**, un outil interactif conçu pour explorer et analyser les sensations à travers une expérience utilisateur ludique et éducative. Ce site, développé dans le cadre d'un atelier par le Cabinet d'Ergothérapie CABEX, permet aux utilisateurs d'évaluer leurs réactions sensorielles (agréables, désagréables, variables ou sans ressenti) face à divers objets et expériences, et de recevoir un profil sensoriel potentiel avec des préconisations adaptées.

## Description
Ce questionnaire est un outil non diagnostique destiné à un usage informatif. Il est idéal pour les ateliers éducatifs ou thérapeutiques, permettant aux participants de mieux comprendre leurs perceptions sensorielles. Les résultats incluent un profil (Hyposensible, Hypersensible, Fluctuant, Adapté ou à explorer) et des suggestions d'aménagements ou d'outils sensoriels, accompagnés d'un bouton pour exporter l'ensemble en PDF.

## Fonctionnalités
- **Évaluation sensorielle** : Répondez à une série de questions sur vos réactions à différents objets et expériences (ex. : élastique de chaise, couverture lestée, bruits forts).
- **Profil sensoriel** : Obtenez un profil basé sur vos réponses (après avoir rempli au moins une réponse).
- **Préconisations** : Accédez à des suggestions personnalisées selon votre profil (aménagements et outils recommandés).
- **Exportation PDF** : Exportez l'ensemble de la page (coordonnées, formulaire, résultats, préconisations) en un fichier PDF avec date et heure.
- **Interface interactive** : Design responsive avec des animations et une mise en page claire.

## Prérequis
- Un navigateur web moderne (Chrome, Firefox, Edge, etc.).
- Une connexion Internet pour charger les bibliothèques externes (`html2canvas` et `jsPDF`).
- Aucun logiciel supplémentaire n'est requis pour l'utilisation en ligne.

## Installation
Ce projet est principalement conçu pour être utilisé directement dans un navigateur via un fichier HTML local ou un serveur web. Suivez ces étapes pour l'exécuter localement :

1. **Téléchargement** :
   - Clonez ou téléchargez ce dépôt : `git clone <URL-du-dépôt>` (si disponible) ou téléchargez le fichier `index.html` et ses dépendances.

2. **Dépendances** :
   - Les bibliothèques `html2canvas` et `jsPDF` sont incluses via des CDN dans le code. Assurez-vous d'avoir une connexion Internet active pour les charger lors du premier accès.

3. **Exécution** :
   - Ouvre le fichier `index.html` dans un navigateur en double-cliquant dessus ou en le servant via un serveur local (recommandé pour éviter les restrictions de sécurité des navigateurs, par exemple avec `python -m http.server` ou un serveur comme XAMPP).

## Utilisation
1. **Accès** :
   - Charge la page `index.html` dans ton navigateur.

2. **Remplissage du questionnaire** :
   - Pour chaque objet ou expérience listée, coche une option (Agréable, Désagréable, Variable, Sans ressenti) et, si applicable, ajoute une sensation ressentie dans le champ texte.

3. **Analyse** :
   - Clique sur le bouton **"Voir mon profil"** pour générer ton profil sensoriel basé sur tes réponses. Assure-toi d'avoir coché au moins une réponse.

4. **Préconisations** :
   - Clique sur **"Voir mes exemples de préconisation"** pour afficher des suggestions adaptées à ton profil.

5. **Exportation** :
   - Après avoir affiché les résultats et les préconisations, clique sur **"Exporter en PDF"** (qui apparaît après les préconisations) pour télécharger un PDF contenant l'ensemble de la page, incluant la date et l'heure d'exportation.

6. **Remarque** :
   - Ce questionnaire n'est pas un diagnostic médical. Consultez un professionnel de santé pour une évaluation approfondie.
